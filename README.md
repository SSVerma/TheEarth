# TheEarth
The project showcases Kotlin Multiplatform Mobile (KMM) app.

> Api: https://countries.trevorblades.com/

> Type: Graphql

> Layers shared: Data layer and Domain layer

> Android Native UI: Jetpack Compose

> iOS Native UI: SwiftUI
